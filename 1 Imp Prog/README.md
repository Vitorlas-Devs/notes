# Programozás

## Notes structure

```graphql
├─ .vscode/* - # VSCode beállítások
├─ [Semester] [Subject name]/* - # tárgyak szemeszterre lebontva
│ └─ [Week number]/* - # minden hét külön mappában
│     ├─ ea/* - # előadás anyagok
│     │ ├─ ea.md - # előadás jegyzet
│     │ └─ ...
│     ├─ gyakorlók/* - # gyakorló feladatok
│     │ └─ [Feladat száma].c
│     ├─ kötelezők/* - # kötelező feladatok
│     │ └─ [Feladat száma].c
│     └─ haladók/* - # haladó feladatok
│       └─ [Feladat száma].c
└─ lib/* - # saját library-k
```
